# ThinkTool

複数の AI CLI（Claude / Codex / Gemini）を使った調査・検討セッションを  
**マインドマップ形式**で管理する Electron デスクトップアプリです。

---

## 概要

- 検討テーマをノードとして階層管理し、各ノードで AI との会話を開始できる
- ノードとプロジェクトフォルダ構造を自動同期
- 会話履歴は CLI 側（`~/.claude/projects/` など）に自動蓄積される
- サブスクリプション範囲内で Claude / Codex / Gemini を利用可能（API 課金不要）

---

## 画面レイアウト

```
┌──────────────┬────────────────────────────┬──────────────────┐
│  左ペイン    │      中央ペイン            │  右ペイン        │
│  ProjectTree │   MindMap（上半分）        │  FileExplorer    │
│              │   NodeTerminal（下半分）   │                  │
│  ─────────── │                            │                  │
│  一時ターミ  │                            │                  │
│  ナル（下）  │                            │                  │
└──────────────┴────────────────────────────┴──────────────────┘
```

### 左ペイン
- プロジェクトツリー（ノード階層表示）
- 最近開いたプロジェクト一覧（最大 16 件）
- 下部に一時ターミナル（汎用シェル、保存なし）

### 中央ペイン
- **マインドマップ**：ノードを図形で表示。右クリックで AI CLI 起動・子ノード追加
- **ターミナル**：xterm.js ベースの内蔵ターミナル（WebGL レンダラー採用）
- 上下境界をドラッグしてサイズ変更可能

### 右ペイン
- ファイルエクスプローラー（プロジェクト未選択時はホームディレクトリ）
- プロジェクト選択時はプロジェクトルートを表示
- アコーディオンボタンで表示・非表示を切り替え可能

## 使い方

### プロジェクトを開く

1. 左ペインの「プロジェクトを開く」ボタンをクリック
2. フォルダを選択すると、そのフォルダがプロジェクトルートになる
3. `tree.json` がなければ自動作成される

### 新規プロジェクトを作成

1. 左ペインの「新規プロジェクト」ボタンをクリック
2. フォルダを選択すると、`tree.json` と初期ノードが作成される

### AI との会話を始める

1. マインドマップまたは左ペインのノードを**右クリック**
2. 「Claude で開く」「Codex で開く」「Gemini で開く」から選択
3. 中央ペイン下部のターミナルで AI が起動する

### 子ノードを追加する

- マインドマップのノードを右クリック → 「子ノードを追加」
- 入力したノード名でサブフォルダが自動作成される

## プロジェクトのフォルダ構成

```
<projectRoot>/
├── tree.json           ← ノードツリーデータ（自動管理）
├── layout.json         ← マインドマップのレイアウト情報（自動管理）
├── <ノード名>/
│   ├── <子ノード名>/
│   └── ...
└── ...
```

ユーザー設定:

```
~/.thinktool/
└── recent.json         ← 最近開いたプロジェクト履歴（最大 16 件）
```
